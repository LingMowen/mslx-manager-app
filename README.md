# MSLX Manager

基于 Vue 3 + TypeScript + Ionic Framework 的 Minecraft 服务器管理应用。

## 功能特性

- 服务器实例管理
- Frp 隧道管理
- 用户权限管理
- 实时控制台
- 文件管理系统
- 备份管理
- 定时任务调度
- 地图可视化

## 技术栈

- Vue 3
- TypeScript
- Ionic Framework
- Vite
- Pinia
- Capacitor

## 开发

```bash
# 安装依赖
npm install

# 开发模式
npm run dev

# 构建
npm run build
```

## APK 构建

```bash
npm run android
```
